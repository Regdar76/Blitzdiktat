# Support

> 🌐 [Deutsch](SUPPORT.md) · **English**

Private project — there is no support channel and no guarantees. GitHub Issues in the (private) repository serve as my own task and bug tracker.

## Self-help checklist

- **Windows:** Does the app start with `run.bat` or `python main.py`? Is `faster-whisper` installed? Microphone permission granted?
- **Android:** Microphone permission granted? For the keyboard: IME enabled and selected? Device offline language pack installed?
- **macOS:** Microphone and accessibility permissions granted?
- **LLM workflows:** OpenAI key stored in the settings and valid (check your quota)?

## Hygiene

Even in the private tracker, do not store anything that does not belong there: no API keys, tokens, private recordings, or confidential transcripts. Security-relevant matters: see [SECURITY.en.md](SECURITY.en.md).
