# Landing Page Brief

Domain: `blitztext.app`

Goal: a very small landing page for an experimental open-source Windows preview.

## Hero

Headline:

> Blitzdiktat for Windows

Subline:

> Speak your thoughts. Turn them into text, cleaner writing, calmer messages, or a meeting protocol.

Body:

> An experimental open-source Windows system tray app. Not finished, not hosted, not plug-and-play. Built to learn from, fork, and improve.

Primary CTA:

> View on GitHub

Secondary CTA:

> Read setup guide

Small line:

> Local transcription included. Bring your own OpenAI API key for rewriting workflows. No hosted Blitzdiktat backend.

## Sections

1. What it does
   - Dictate
   - Improve
   - Calm down
   - Add emojis
   - Create meeting protocols

2. How it works
   - Clone and install with pip
   - Paste your own OpenAI API key (optional — local transcription works without one)
   - Press a hotkey and speak
   - Get text back in the active window

3. Open-source preview
   - Windows only
   - MIT License
   - experimental
   - no warranty
   - local transcription via faster-whisper, no GPU required

4. Privacy, plainly
   - local transcription stays on your PC (faster-whisper, no network)
   - online rewriting workflows send text to OpenAI
   - no public Blitzdiktat backend
   - do your own review before sensitive use

5. Roadmap
   - easier first-run setup
   - packaged Windows installer
   - local text rewriting
   - better protocol templates
   - community issues

## Do Not Promise

- fully offline use for all workflows (rewriting still uses OpenAI)
- production readiness
- hosted service
- free usage without API costs for rewriting
- guaranteed support
- other platforms
- bundled local models

## Visual Direction

Use a real Windows screenshot or short demo GIF. Keep the page calm, sparse, and honest. Avoid fake metrics, oversized SaaS claims, and corporate origin story.
